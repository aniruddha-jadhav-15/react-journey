# 01 — React Setup

## What is React?

React is a JavaScript library developed by Facebook.
It helps build and manage large scale applications
using reusable components.

## Why Vite over Create React App?

- Create React App is OLD and slow
- React no longer recommends Create React App
- Vite is faster, lightweight and modern
- Industry standard in 2024+

## Create React App with Vite:

npm create vite@latest app-name -- --template react
cd app-name
npm install
npm run dev

## Key Files:

- main.jsx → entry point
- App.jsx → root component
- index.html → single HTML file
