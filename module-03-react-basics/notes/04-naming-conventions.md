# 04 — Naming Conventions

## 3 Naming Conventions in React:

### 1. kebab-case

Used for: file names, CSS classes
Example: my-component.jsx, user-profile.css

### 2. PascalCase

Used for: Component names
Example: UserProfile, TodoList, NavBar

### 3. camelCase

Used for: variables, functions, props
Example: getUserData(), firstName, onClick

## Quick Reference:

File names → kebab-case
Components → PascalCase
Variables/Fns → camelCase
CSS classes → kebab-case
