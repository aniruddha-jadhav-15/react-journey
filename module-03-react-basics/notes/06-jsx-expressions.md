# JSX Expressions

## What is JSX Expression?

Using JavaScript inside JSX with {} curly braces

## What can go inside {}?

- Variables
- Math expressions
- Function calls
- Ternary operator
- Template literals

## What CANNOT go inside {}?

- if/else statements
- for loops
- switch statements
  (Only expressions, not statements!)

## Syntax:

const name = "Aniruddha";

<h1>{name}</h1>

## Example:

<p>{age > 18 ? "Adult" : "Minor"}</p>
