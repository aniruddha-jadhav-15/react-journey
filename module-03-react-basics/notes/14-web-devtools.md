# 14 — Web DevTools

## What is Web DevTools?

Browser built-in tools for developers
to inspect, debug and optimize web apps.

## How to open:

Right click → Inspect
OR
F12 (Windows)
CMD + Option + I (Mac)

## Important Tabs:

### Elements Tab:

→ Inspect HTML structure
→ Edit CSS live
→ See DOM changes

### Console Tab:

→ Check errors and warnings
→ Run JavaScript
→ console.log() output shows here

### Network Tab:

→ See all API calls
→ Check request/response
→ Debug loading issues
→ See status codes (200, 404, 500)

### Sources Tab:

→ Debug JavaScript
→ Add breakpoints
→ Step through code line by line

### React DevTools (Extension):

→ Install from Chrome Extensions
→ See component tree
→ Inspect props and state
→ Debug re-renders

## Most used in React:

Console → check errors
Network → debug API calls
React tab → inspect components
