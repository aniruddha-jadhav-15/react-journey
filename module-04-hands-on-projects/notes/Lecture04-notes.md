# Lecture 04 — Header Structure & Styling

## 📌 What I Learned

- How to style a component to match Figma design
- How to use flexbox to align header elements properly
- How to make tag buttons look like pills

## 🧱 Structure

- TaskForm component has its own CSS file
- Each element targeted by its className
- Header layout built using flexbox

## 💡 Key Takeaways

- Always match your CSS to the Figma design
- Flexbox is the best way to align
  header elements in a row
- Consistent spacing and sizing makes
  UI look professional
- Component level CSS keeps styles
  isolated and maintainable
