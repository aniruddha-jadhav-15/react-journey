# Lecture 05 — Creating a Reusable Tag Component

## 📌 What I Learned

- Why we create reusable components
- How to create a Tag component

## 🧩 Why Reusable Components

- Tag is used multiple times in the app
- Instead of repeating same code we
  create one component and reuse it
- Makes code clean and maintainable

## ⚠️ Current Limitation

- Tag text is hardcoded as DEV right now
- Next lecture we use props to make
  it truly dynamic and reusable

## 💡 Key Takeaway

- If same UI is used multiple times
  always create a separate component
- Reusability is one of the core
  strengths of React
