## Lecture 04 — Client Server Architecture

Client (browser) sends HTTP request to server.
Server processes and sends response back.

HTTP Methods:
GET → read data
POST → create data
PATCH → update part of data
DELETE → remove data

Status Codes:
200 → success
404 → not found
500 → server error

💡 Key Takeaway
As frontend dev you need to know which HTTP
method to use for each type of operation.
